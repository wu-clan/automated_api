## 设计思路
- unittest + 关联 + excle + ddt

## 目录结构介绍
- bin：可执行文件，程序入口
- conf：配置文件，各种路径配置、ip、端口等
- data：测试数据excel
- lib：工具库
- reprot：测试报告
- test_case：测试用例
- log：日志文件
- README.md：说明文件
